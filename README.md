# Drug Interaction Monitor

AI-powered drug interaction safety analysis built for the **Arize @ Google Cloud Partnerships Hackathon**. Select medications and get a structured 5-section safety report generated by Gemini 2.5 Flash, fully traced in Arize Phoenix.

**Live demo:** https://drug-interaction-agent-133915220356.us-east1.run.app

## Pipeline

```
resolve_all_medications   ->  NIH RxNorm (drug name -> RxCUI)
    |-- check_all_interactions  ->  NIH Drug Interaction API (every pair)
    |-- get_fda_enrichment      ->  FDA Labels + OpenFDA FAERS
synthesize_safety_report  ->  Gemini 2.5 Flash (5-section report)
evaluate_report_quality   ->  LLM-as-a-Judge (criteria from Phoenix Prompt Management)
apply_self_improvement    ->  Rewrites low-scoring sections, writes scores to Phoenix dataset
```

Sections averaging below 0.85 across past runs receive focused prompting on the next run.

All Gemini calls are auto-instrumented via `openinference-instrumentation-google-genai` — prompts, responses, and token counts appear as LLM spans in Phoenix traces.

## Phoenix MCP

Four operations appear as MCP tool calls in Phoenix traces:

| Tool | Purpose |
|---|---|
| `get-prompt-by-identifier` | Fetch versioned judge criteria |
| `upsert-prompt` | Sync updated criteria when code changes |
| `get-dataset-examples` | Read historical scores to guide synthesis |
| `add-dataset-examples` | Write this run's scores for future runs |

## Quickstart

**Prerequisites:** Python 3.10-3.12, [uv](https://docs.astral.sh/uv/), Node.js 18+, a Phoenix Cloud API key at [app.phoenix.arize.com](https://app.phoenix.arize.com), and Google auth via `GOOGLE_API_KEY` or Vertex AI.

```bash
git clone <repo> && cd drug-interaction-agent
cp .env.example .env
# Fill in PHOENIX_API_KEY, PHOENIX_COLLECTOR_ENDPOINT, and GOOGLE_API_KEY or Vertex settings
uv sync
make ui                # opens http://localhost:8501
```

```bash
make run MESSAGE="I take warfarin, aspirin, and clopidogrel"   # CLI with full trace
make experiment NAME="v1"                                      # Phoenix benchmark
make deploy                                                    # deploy to Cloud Run
```

## Layout

| Path | Purpose |
|---|---|
| `agent/app.py` | Streamlit web UI |
| `agent/main.py` | CLI runner |
| `agent/phoenix_client.py` | Phoenix MCP session, dataset/prompt ops, SDK fallbacks |
| `agent/instrumentation.py` | Phoenix OTel tracing + OpenInference GoogleGenAI auto-instrumentation |
| `agent/run_experiment.py` | Phoenix Experiments benchmark |
| `agent/drug_interaction_agent/` | Pipeline tools, prompts, Pydantic models, ADK agent |
| `.env.example` | All required environment variables with descriptions |
| `Dockerfile` | Python 3.11 + Node.js 20 for Phoenix MCP subprocess |
